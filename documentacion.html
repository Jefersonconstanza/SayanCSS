📘 Documentación de Saiyan CSS

Esta página incluye ejemplos interactivos y explicaciones de cómo usar los estilos personalizados de tu librería Saiyan CSS, inspirada en el universo de Dragon Ball.

🔘 Botones personalizados

Usá la clase .boton junto con .boton-goku, .boton-vegeta, etc., para aplicar estilos según el personaje. 

<button class="boton boton-goku">Botón Goku</button>

🌑 Modo Vegeta (Modo Oscuro)

Saiyan CSS incluye un modo oscuro personalizado inspirado en Vegeta. Cuando se activa, se modifican colores de fondo, textos, botones y tarjetas, y se muestra la imagen de Vegeta en lugar de Gokú.

🧠 ¿Cómo funciona?

  • Al pulsar el botón "Modo Vegeta", se agrega la clase .modo-vegeta al <body>.
  • Se guarda en localStorage para que el modo persista al recargar la página.

📌 Ejemplo de uso


<button class="boton-modo" onclick="activarModoVegeta()">
  Modo Vegeta
</button>

🌞 Modo Gokú (Modo Claro)

El Modo Gokú restaura la interfaz a su estética original inspirada en Gokú, eliminando la clase .modo-vegeta del <body>.

🧠 ¿Cómo funciona?

  • Al pulsar el botón "Modo Gokú", se remueve la clase .modo-vegeta del <body>.
  • Esto restaura los estilos predeterminados y elimina la configuración guardada en localStorage.

📌 Ejemplo de uso


<button class="boton-modo" onclick="activarModoGoku()">
  Modo Gokú
</button>

📣 Alertas temáticas

Las alertas se activan con la clase .alerta y variantes como .alerta-exito o .alerta-peligro para distintos mensajes.

¡Esta es una alerta de éxito Saiyajin!
¡Frieza se acerca! Esto es una alerta de peligro.
<div class="alerta alerta-exito">¡Mensaje exitoso!</div>

📋 Formularios estilizados

Estilizá tus formularios usando las clases .formulario, .entrada-formulario, .seleccion-formulario y .etiqueta-formulario. 

<form class="formulario">
  <input type="text" class="entrada-formulario">
</form>

🏷️ Insignias (Badges)

Utilizá la clase .insignia junto a variantes como .insignia-goku, .insignia-vegeta, etc., para mostrar etiquetas visuales.

Saiyajin Orgullo Namekusein Villano 
<span class="insignia insignia-goku">Saiyajin</span>

⚡ Animación de transformación

Agregá la clase .transformado a cualquier elemento para aplicar una animación de "carga de ki". 

<button class="boton transformado">¡Transformarse!</button>

🛠️ Clases Utilitarias

Estas clases permiten modificar rápidamente tamaño, texto, márgenes, sombras, colores y más.

Texto grande (.txt-grande)

Texto mediano (.txt-mediano)

Texto chico (.txt-chico)

Texto Goku en negrita (.txt-goku .txt-bold)

Texto Vegeta en itálica (.txt-vegeta .txt-italic)

Texto Piccolo (.txt-piccolo)
